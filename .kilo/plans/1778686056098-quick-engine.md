# Update Treatment Result Handling

The API response for `getTreatment` has changed. It now returns `title` and `description` instead of `treatmentPlan` and `recommendations`.

## Changes

### 1. Update Models
- Modify `src/app/Models/photo.models.ts` to update the `TreatmentResult` interface to match the new API response:
  - Remove `diseaseId`, `treatmentPlan`, and `recommendations`.
  - Add `title: string` and `description: string`.

### 2. Update Upload Image Component
- Modify `src/app/Components/upload-image-component/upload-image-component.ts`:
  - Update the mapping of `TreatmentResult` in `analyzeImage` to use `title` and `description`.
  - Set `status` to `treatment.title` and `recommendation` to `treatment.description`.

## Verification
- Run the application and upload an image that triggers a disease prediction.
- Verify that the disease title and treatment description are correctly displayed.
